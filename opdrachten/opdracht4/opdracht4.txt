Front-end & back-end frameworksWat is het verschil tussen front end back end?Een front end developer is verantwoordelijk voor wat de gebruiker visueel ziet in de webbrowser of aplicatie.Een back end developer werkt aan de communicatie tussen een database en een webbrowser. Hij doet dit aan de hand van servers.Geef 5 voorbeelden van front end framework.- HTML- CSS- Javescript- JQuery- SilverlightGeef 5 voorbeelden van back end frameworks.- Java- PHP- Ruby on Rails- Python- .NetWelke zijn de populairste back end en front end frameworks?Back end: #1 Java; #2 C; #3 C++Front end: #1 Javascript; #2 CSS; #3 HTML
